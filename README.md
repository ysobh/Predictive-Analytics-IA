# Predictive-Analytics-IA
3-class classification model predicting Premier League match outcomes (Home Win / Draw / Away Win) using rolling form data from the 2024/25 season.

## Setup
```bash
pip install -r requirements.txt
```
## Data

Download the dataset from Kaggle: https://www.kaggle.com/datasets/furkanark/premier-league-2024-2025-data
 
Place the contents in a folder named `PL_DATA/` in the project root.
 
## Running
 
Open and run `Assignment.ipynb` from top to bottom. All cells must be run in order.
 
## Notes
 
- Python 3.10+
- No GPU required
- Expected runtime: under 5 minutes
